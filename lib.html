   do
  {
      digit = num % 10;
      rev = (rev * 10) + digit;
     num = num / 10;

  }   while (num != 0);
cout << "The reverse of the number is:" << rev << endl;

  if (n == rev)

HTML Multiple choice questions (MCQ's)


1.   HTML stands for -
a. HighText Machine Language
b. HyperText and links Markup Language
c. HyperText Markup Language
d. None of these
ANSWER: (c) HyperText Markup Language


2. The correct sequence of HTML tags for starting a webpage is -
a. Head, Title, HTML, body
b. HTML, Body, Title, Head
c. HTML, Head, Title, Body
d. HTML, Head, Title, Body
ANSWER: (d) HTML, Head, Title, Body


3. Which of the following attribute is used to provide a unique name to an element?
a. class
b. id
c. type
d. None of the above
ANSWER: (b) id


4. Which of the following HTML attribute is used to define inline styles?
a. style
b. type
c. class
d. None of the above
ANSWER: (a) style
